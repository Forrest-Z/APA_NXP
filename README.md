# APA

#### 项目介绍
基于NXP 的Power PC架构微处理器MPC5744的自动泊车系统设计
##### 平行泊车
###### 一次入库
如下图所示，是平行车位一次入库的情形。当车位长度足够时，可以规划一段曲线，使车辆一次入库。
![平行泊车一次入库情形](./Doc/GIF/平行一次入库.gif)
###### 多次尝试入库
如下图所示，是平行车位多次尝试入库的情形。当车位长度不满足一次入库条件时，可以在库内多次尝试，使车辆最终入库。
![平行泊车多次尝试情形](./Doc/GIF/平行多次尝试.gif)
